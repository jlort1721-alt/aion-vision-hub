# AION Vision Hub — Reboots Module

## Overview
Guided device restart and recovery management with AION Agent assistance.

## Data Model
- `reboot_tasks`: Task registry with device, reason, status, result, recovery time

## Features
- Device listing with reboot eligibility
- Section-based organization
- Guided restart procedures (step-by-step)
- AION Agent diagnostic suggestions
- Status tracking: pending, in-progress, completed, failed
- Recovery time measurement
- Before/after health comparison (conceptual)

## Integration with Live View
- Alert badges for failing devices
- One-click reboot initiation from camera cells
- AION-assisted troubleshooting inline

## Procedures
1. Identify failing device
2. AION suggests diagnosis
3. Operator initiates guided reboot
4. System tracks recovery
5. Log result and recovery time
