# AION Vision Hub — Installation Guide (PWA)

AION Vision Hub is a Progressive Web App (PWA) that can be installed on any device for a native app experience.

> **Note**: PWA installation requires HTTPS. Until a domain with SSL is configured, the app can be used in the browser at `http://204.168.153.166/` but cannot be installed as a PWA.

---

## Windows (Chrome)

1. Open the app URL in **Google Chrome**
2. Look for the install icon (⊕) in the address bar, or click **⋮** → **"Install AION Vision Hub..."**
3. Click **"Install"** in the dialog
4. The app appears in your **Start Menu** and **Desktop**
5. It runs in its own window (no browser UI)

## Windows (Edge)

1. Open the app URL in **Microsoft Edge**
2. Click **"..."** → **"Apps"** → **"Install this site as an app"**
3. Click **"Install"**
4. The app appears in Start Menu

## Mac (Chrome)

1. Open the app URL in **Google Chrome**
2. Click **⋮** → **"Install AION Vision Hub..."**
3. Click **"Install"**
4. The app appears in **Applications** folder and **Launchpad**

## Mac (Safari — macOS Sonoma 14+)

1. Open the app URL in **Safari**
2. Click **File** → **"Add to Dock"**
3. The app appears in your Dock as a standalone app

## iPhone / iPad (Safari)

1. Open the app URL in **Safari**
2. Tap the **Share** button (↑)
3. Scroll down and tap **"Add to Home Screen"**
4. Tap **"Add"**
5. The app icon appears on your Home Screen

## Android (Chrome)

1. Open the app URL in **Chrome**
2. Tap the **"Add to Home Screen"** banner (or ⋮ → "Install app")
3. Tap **"Install"**
4. The app icon appears on your Home Screen

---

## Features When Installed

| Feature | Browser | Installed PWA |
|---------|---------|---------------|
| Standalone window | No | Yes |
| App icon on desktop | No | Yes |
| Push notifications | Limited | Full support |
| Offline access | Partial | Full cached content |
| Auto-updates | Instant | On next launch |
| Keyboard shortcuts | Standard | Full app shortcuts |

## Updating the App

The PWA auto-updates when a new version is deployed. When an update is detected:
1. A notification bar appears: "New version available"
2. Click "Update" to reload with the latest version
3. No manual action needed — updates are automatic

## Uninstalling

- **Windows/Mac**: Right-click app icon → "Uninstall"
- **Chrome**: chrome://apps → Right-click → "Remove"
- **iPhone**: Long-press icon → "Remove App"
- **Android**: Long-press icon → "Uninstall"
