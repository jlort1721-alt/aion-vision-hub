# AION Vision Hub — Intercom IP Module

## Overview
Centralized IP intercom management with Voice AI and WhatsApp integration.

## Data Model
- `intercom_devices`: Fanvil/SIP devices per section
- `intercom_calls`: Call logs with direction, duration, attendant type

## Features
- 50 sections with device assignment
- Device health monitoring
- Call history and logs
- Quick call actions
- Attendant mode selection: AI Voice, Human Operator, Mixed

## Voice AI (Planned)
- ElevenLabs integration for AI voice responses
- Configurable welcome messages per section
- Visitor validation via database lookup
- Natural conversation flow

## WhatsApp Integration (Planned)
- Message panel with conversation threads
- AI-powered auto-responses
- Quick response templates
- Automation flows per section

## Integration with Live View
- Quick call buttons per section
- Welcome message triggers
- AI/operator mode toggle
- Call status indicators
