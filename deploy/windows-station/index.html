Windows — Instrucciones

1
Descargar Install-AionStation.ps1
2
Click derecho sobre el archivo → "Ejecutar con PowerShell"
3
Si pide permisos de administrador, aceptar
4
Reiniciar cuando lo solicite
# Opciones avanzadas (PowerShell como Admin): .\Install-AionStation.ps1 -DualMonitor # Segundo monitor .\Install-AionStation.ps1 -KioskMode # Pantalla completa .\Install-AionStation.ps1 -StationName "CCTV-01" # Nombre personalizado

macOS — Instrucciones

1
Abrir Terminal
2
Ejecutar el comando de abajo
curl -sL https://aionseg.co/installer/Install-AionStation-Mac.sh | bash
# Con opciones: DUAL_MONITOR=true STATION_NAME="CCTV-MAC-01" bash Install-AionStation-Mac.sh
